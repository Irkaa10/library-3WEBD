# Library 3WEBD

## Getting Started

1. **Navigate to the project directory:**
   ```bash
   cd library-3WEBD
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. Open your browser and go to [http://localhost:3000](http://localhost:3000) to view the app.
